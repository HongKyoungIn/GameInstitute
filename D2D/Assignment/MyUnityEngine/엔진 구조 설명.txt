엔진 구조 설명
1. Engine은 정적 라이브러리로 구성되며 클라이언트에서 게임을 만들기 위한 다양한 기능들을 보유하고 있습니다.
2. Client는 Engine을 참조하는 windows Desktop Application 입니다.

[Engine]
1. Core
- 프로그램을 실행하기 위한 클래스입니다.
- 윈도우 창을 등록하고 생성합니다.

2. GameObject
- 유니티에서 사용하는 GameObject 역할을 수행합니다.
- 각종 컴포넌트들을 추가하여 동작하는 객체를 생성합니다.

3. Component
- 유니티의 Component 역할을 수행합니다.
- 미리 만들어진 부품들이며 다양한 기능들을 만듭니다.

4. Renderer
- RenderFactory 생성
- RenderTarget 생성
- d2d 관련한 초기화 기능 수행

[Clinet]
1. Scene1
- 게임을 만들기 위한 코드를 작성하는 클래스입니다.
- Core를 상속받아 필요한 오브젝트를 생성하고 엔진에 있는 컴포넌트들을 이용하여 게임을 만듭니다.

2. Main
- 게임을 실행시키는 메인 함수입니다.
